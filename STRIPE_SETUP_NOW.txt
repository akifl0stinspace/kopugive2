STEP 1: Install Stripe PHP SDK (2 minutes)
-------------------------------------------
Open Command Prompt and run:

    cd C:\xampp\htdocs\kopugive
    composer install

If you don't have Composer:
‚ö†Ô∏è  STOP! Read this first: INSTALL_COMPOSER_FIRST.md
    Or download from: https://getcomposer.org/Composer-Setup.exe
    Takes 5 minutes to install, then come back here!


STEP 2: Create Stripe Account (2 minutes)
------------------------------------------
1. Go to: https://dashboard.stripe.com/register
2. Sign up with your email
3. You'll be in TEST MODE automatically (perfect!)


STEP 3: Get Your API Keys (1 minute)
-------------------------------------
1. In Stripe Dashboard, go to: Developers ‚Üí API Keys
2. You'll see two keys:
   - Publishable key (starts with pk_test_)
   - Secret key (starts with sk_test_) - Click "Reveal test key"
3. Copy both keys


STEP 4: Update Configuration (1 minute)
----------------------------------------
1. Open: config/config.php
2. Find these lines (around line 42-44):

   define('STRIPE_PUBLISHABLE_KEY', 'pk_test_YOUR_PUBLISHABLE_KEY_HERE');
   define('STRIPE_SECRET_KEY', 'sk_test_YOUR_SECRET_KEY_HERE');

3. Replace with your actual keys from Step 3


STEP 5: Run Database Migration (30 seconds)
--------------------------------------------
OPTION A (Easiest):
   Visit: http://localhost/kopugive/install_stripe.php
   Click: "Run Database Migration" button

OPTION B (Manual):
   1. Open phpMyAdmin
   2. Select 'kopugive' database
   3. Click 'SQL' tab
   4. Copy and paste this:

      ALTER TABLE donations 
      ADD COLUMN stripe_payment_intent_id VARCHAR(255) NULL AFTER transaction_id,
      ADD COLUMN stripe_checkout_session_id VARCHAR(255) NULL AFTER stripe_payment_intent_id,
      ADD COLUMN payment_status VARCHAR(50) DEFAULT 'pending' AFTER status;
      
      UPDATE donations SET payment_status = 'pending' WHERE payment_status IS NULL;

   5. Click 'Go'

================================================================================
  TROUBLESHOOTING
================================================================================

"Composer not found"
  ‚Üí Download: https://getcomposer.org/download/
  ‚Üí Install and restart terminal

"Class 'Stripe\Stripe' not found"
  ‚Üí Run: composer install
  ‚Üí Check vendor/ folder exists

"Invalid API key"
  ‚Üí Check config.php has correct keys
  ‚Üí No extra spaces or quotes
  ‚Üí Keys start with pk_test_ and sk_test_

Payment succeeds but not verified
  ‚Üí Check database migration was run
  ‚Üí Check PHP error logs: logs/php_errors.log
  ‚Üí Verify API keys are correct

FPX not showing up
  ‚Üí Check STRIPE_CURRENCY is set to 'myr'
  ‚Üí Ensure you're in test mode
